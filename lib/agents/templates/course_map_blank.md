| Domain | Level (low → high) | Sample verbs |
|---|---|---|
| **Cognitive** | Remember → Understand → Apply → Analyze → Evaluate → Create | define, explain, apply, analyze, evaluate, create |
| **Affective** | Receiving → Responding → Valuing → Organizing → Characterizing | attend, comply, accept, integrate, embody |
